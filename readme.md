# Pairs
### Гра "Пари"

Для гри в "Пари" необхідно розпочати гру, після чого буде наданий час для запам'ятовування карток (близько 5 секнд), по закінченню якого картки будуть перевернуті. Далі необхідно натиснути на дві однакові картки і продовжувати до тих пір доки всі картки не будуть відгадані або незакінчиться час для відгадування (близько 15 секунд).
